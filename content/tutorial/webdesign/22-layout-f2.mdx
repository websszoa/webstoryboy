# 소개

안녕하세요! 웹스토리보이입니다 😊

이번 시간에는 **F-2 유형 레이아웃**을 함께 만들어보겠습니다. 전체적인 구조는 F-1과 유사하지만, 콘텐츠 배치 방식에서 약간의 차이가 있어요. 큰 틀은 익숙하니 가볍게 복습하는 느낌으로 천천히 따라와 주세요!

## 1. 기본 구조 만들기

VSCODE를 실행하고 webdesign 폴더 안에 layoutF-2.html파일을 만들겠습니다.

!를 치고 tab버튼을 누르면 다음과 같이 나타납니다. lang는 ko로 변경하고 title은 웹디자인개발기능사 레이아웃 F-2으로 변경해주겠습니다.

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>웹디자인개발기능사 레이아웃 F-2</title>
</head>
<body>

</body>
</html>
```

전체 구조는 `#wrap`이라는 부모 요소 안에 네 개의 주요 섹션으로 구성되어 있습니다. 상단에는 `<header>`가 위치하고, 그 아래로 `<slider>`, 메인 콘텐츠를 담는 `<section id="contents">`, 마지막으로 하단의 `<footer>`가 차례로 배치되어 있습니다.

```html
<body>
    <div id="wrap">
        <header id="header"></header>
        <article id="slider"></article>
        <section id="contents"></section>
        <footer id="footer"></footer>
    </div>
</body>
```

```css
* {
    margin: 0;
    padding: 0;
}
#wrap {
    width: 100%;
}
#header {
    width: 1340px;
    height: 100px;
    margin: 0 auto;
    background-color: #efefef;
}
#slider {
    width: 100%;
    height: 350px;
    background-color: #e3e3e3;
}
#contents {
    width: 1340px;
    height: 450px;
    margin: 0 auto;
    background-color: #d9d9d9;
}
#footer {
    width: 1340px;
    height: 120px;
    margin: 0 auto;
    background-color: #d1d1d1;
}
```

## 2. 각 섹션 작업하기

`<header>` 영역은 웹페이지의 상단, 즉 머리 부분을 담당하는 구역입니다. 이 안에는 왼쪽에 위치할 **로고** 영역과, 오른쪽에 배치될 **내비게이션 메뉴(nav)**가 포함되어 있어요.

```html
<header id="header">
    <h1 class="logo"></h1>
    <nav class="nav"></nav>
</header>
<!-- //header -->
```

```css
#header {
    width: 1340px;
    margin: 0 auto;
    display: flex;
}
#header .logo {
    width: 20%;
    height: 100px;
    background-color: #efefef;
}
#header .nav {
    width: 80%;
    height: 100px;
    background-color: #e3e3e3;
}
```

`<slider>` 영역은 이미지 슬라이드나 배너가 들어가는 공간입니다. 전체 너비(`width: 100%`)를 차지하며, 높이는 `350px`로 설정되어 있어요. 하단에는 `.link`라는 박스가 하나 들어가 있는데, 이 박스는 **슬라이더 영역 안의 하단에 가로 중앙**에 위치하도록 CSS로 정렬되었습니다. `position: absolute`와 `left: 50%`, 그리고 `transform: translateX(-50%)` 조합은 정중앙 정렬 공식처럼 자주 쓰이는 패턴이니 꼭 기억해두세요! 😉

```html
<article id="slider">
    <div class="link">링크</div>
</article>
<!-- //slider -->
```

```css
#slider {
    width: 100%;
    height: 350px;
    background-color: #d9d9d9;
    position: relative;
}
#slider .link {
    width: 1340px;
    height: 100px;
    background-color: #c7c7c7;
    position: absolute;
    left: 50%;
    bottom: 0;
    transform: translateX(-50%);
}
```

`<section id="contents">`는 본문 영역의 핵심 콘텐츠를 담는 공간이에요. 이 안에는 **갤러리**와 **공지사항** 두 개의 `<article>` 박스가 포함되어 있습니다. (배너 아래에 공지·갤러리를 감싼 div로 좌우 분할 구조입니다.)

```html
<section id="contents">
    <article class="banner"></article>
    <div>
        <article class="notice"></article>
        <article class="gallery"></article>
    </div>
</section>
<!-- //contents -->
```

```css
#contents {
    width: 1340px;
    margin: 0 auto;
}
#contents .banner {
    width: 100%;
    height: 150px;
    background-color: #b8b8b8;
}
#contents > div {
    width: 100%;
    display: flex;
}
#contents > div .notice {
    width: 50%;
    height: 350px;
    background-color: #b1b1b1;
}
#contents > div .gallery {
    width: 50%;
    height: 350px;
    background-color: #a3a3a3;
}
```

`<footer>`는 페이지의 맨 아래 영역이며, 좌우 2개 + 가운데 2단 구성으로 이루어진 **3분할 + 중첩 구조**입니다. `.footer1`은 좌측, `.footer3`은 우측에 배치되고, `.footer2`는 가운데에서 `.footer2-1`, `.footer2-2`로 위아래로 나뉩니다.

```html
<footer id="footer">
    <div class="footer1"></div>
    <div class="footer2">
        <div class="footer2-1"></div>
        <div class="footer2-2"></div>
    </div>
</footer>
<!-- //footer -->
```

```css
#footer {
    width: 1340px;
    margin: 0 auto;
    display: flex;
}
#footer .footer1 {
    width: 80%;
    height: 120px;
    background-color: #9d9d9d;
}
#footer .footer2 {
    width: 20%;
}
#footer .footer2 .footer2-1 {
    width: 100%;
    height: 60px;
    background-color: #929292;
}
#footer .footer2 .footer2-2 {
    width: 100%;
    height: 60px;
    background-color: #838383;
}
```

## 3. 정리

### ✅ 주요 HTML 태그

- `<header>` : 상단 영역으로, 로고와 내비게이션 메뉴를 포함합니다.
- `<article id="slider">` : 슬라이드 영역으로 전체 너비를 차지하며, 링크 박스가 포함됩니다.
- `<section id="contents">` : 콘텐츠 영역으로 배너, 공지사항, 갤러리를 포함한 분할 구조입니다.
- `<footer>` : 사이트 하단으로 좌우 2분할 + 중앙 2단 구조로 되어 있습니다.

### ✅ 주요 CSS 속성

- `width: 1340px + margin: 0 auto` → 고정형 레이아웃을 중앙 정렬하는 방식입니다.
- `position: absolute + left: 50% + transform` → 슬라이드 안 링크 박스를 정확히 가로 중앙에 배치합니다.
- `display: flex` → 공지사항과 갤러리를 가로로 나란히 배치하는 데 사용됩니다.
- `background-color` → 시각적으로 영역을 구분하고 레이아웃 구조를 쉽게 파악할 수 있도록 도와줍니다.

### ✅ 이번 실습의 핵심

- 고정형 콘텐츠(1340px)와 유동형 영역(slider)의 조합으로 다양한 배치 방식을 연습했습니다.
- `position`과 `transform`을 이용한 **가로 정중앙 배치 기법**을 익혔습니다.
- **콘텐츠 영역의 상단/하단 분할** 및 **공지/갤러리 좌우 분할 구조**를 연습했습니다.
- 푸터의 **2분할 + 내부 2단 구조** 구현을 통해 중첩 레이아웃을 구성하는 법을 익혔습니다.
