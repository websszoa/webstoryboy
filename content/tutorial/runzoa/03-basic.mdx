# 기본 사용법

프로젝트 구조와 자주 쓰는 명령어를 알아봅니다.

## 프로젝트 구조

- `app/` — Next.js App Router 페이지
- `components/` — 재사용 컴포넌트
- `lib/` — 유틸·설정
- `content/` — 마크다운/MDX 콘텐츠

## 자주 쓰는 명령어

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```

### 프로덕션 실행

```bash
npm run start
```
