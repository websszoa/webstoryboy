# 환경 설정

프로젝트를 시작하기 전에 필요한 도구를 설치하고 설정합니다.

## Node.js 설치

1. [Node.js 공식 사이트](https://nodejs.org/)에서 LTS 버전을 다운로드합니다.
2. 설치 후 터미널에서 다음 명령으로 확인합니다.

```bash
node -v
npm -v
```

## 프로젝트 클론

```bash
git clone https://github.com/your-org/runzoa.git
cd runzoa
npm install
```

## 환경 변수 설정

`.env.local` 파일을 생성하고 필요한 값을 입력합니다.

| 변수명 | 설명 |
|--------|------|
| `NEXT_PUBLIC_SUPABASE_URL` | Supabase 프로젝트 URL |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Supabase anon key |

다음 문서에서는 **기본 사용법**을 다룹니다.
